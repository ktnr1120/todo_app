# TODO App

javascriptで作成したtodoアプリ。
学習用として作成。

## 目的

javascriptを使用して、フロントエンドとバックエンドの両方を実装することを目的としている。
フロントエンドはReactを使用。バックエンドはNode.jsを使用している。
また、AIを使用しての開発で、実際にwebアプリの開発経験がない状態で、どこまで自走できるかを試した。

## 着地点

3日目安でwebアプリの開発を完了させる。
フロントエンドとバックエンドの両方を実装させる。
設計は自分、開発等はAIに任せる。

## 達成

3日での開発はおおむね終了。
フロントエンドとバックエンドの両方を実装させることができた。
設計は自分で行い、AIに開発をさせる方針がうまくいった。

## 反省点

フロント側の設計書が漏れていた。
HTTP通信が初めてだったので、APIの設計が不十分だった。
javascriptは開発経験がないので、コードの確認が後手に回ってしまった。
gitignoreの存在を把握していなかったため、node_modulesや.envファイルがコミットされてしまった。
設計段階でフォルダ構成を考慮すべきだった。

## 今後の課題

反省点を取り入れて次回の開発に活かす。
次はログイン機能の実装を考えているのでセキュリティ面や攻撃への対策も考慮する。
